## 2. Структура сайта

### Сайт: [Red Collar](https://redcollar.ru/)

#### Скриншот страницы:
![Red Collar Screenshot](img\screensite.png)

#### Описание структуры:
1. **Шапка (Header):**
   - Логотип, навигационное меню, кнопка "Связаться".
   - Используется `position: fixed;` для фиксации шапки вверху страницы.
2. **Основная часть (Main):**
   - Секции с проектами, информация о студии, услуги.
   - Используется `display: flex;` для выравнивания элементов.
3. **Подвал (Footer):**
   - Контактная информация, ссылки на социальные сети.
   - Используется `grid` для создания сетки.

#### Использование методологий:
- **Методология именования классов:** BEM (Block-Element-Modifier).
- **HTML5 теги:** Используются `<header>`, `<main>`, `<footer>`, `<section>`.
- **Обертки `div`:** Минимальное количество, структура понятна.
- **Исходный код:** Легко читаем, хорошо структурирован.

## 3. Реализация блока

### Блок: Карточка проекта на [Red Collar](https://redcollar.ru/)

#### Скриншот блока:
![Red Collar Card Screenshot](img\block.png)

#### Пошаговое описание:
1. Создайте контейнер для карточки с классом `.project-card`.
2. Добавьте изображение с классом `.project-card__image` и установите `object-fit: cover;`.
3. Внутри контейнера создайте блок для текста с классом `.project-card__content`.
4. Добавьте заголовок с классом `.project-card__title` и стилями для шрифта.
5. Добавьте описание с классом `.project-card__description` и установите `font-size: 14px;`.
6. Создайте блок для кнопки с классом `.project-card__button` и стилями для фона и текста.
7. Установите `box-shadow` для контейнера карточки для создания эффекта тени.
8. Добавьте `border-radius: 8px;` для скругления углов карточки.
9. Установите `transition` для плавного изменения стилей при наведении.
10. Добавьте `:hover` эффект для изменения `box-shadow` и `transform: scale(1.05);`.
11. Используйте `grid` для размещения карточек в сетке.
12. Установите `gap` между карточками для отступов.
13. Добавьте медиа-запросы для адаптации карточек под мобильные устройства.

#### Свойства:
- `object-fit: cover;`
- `flexbox`
- `box-shadow`
- `border-radius`
- `transition`
- `transform`
- `grid`
- `gap`
